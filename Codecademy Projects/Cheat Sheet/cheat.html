HTML:

Elements and Structure

Tag Name Description
a Anchor Used to create hyperlinks in an HTML document.
head Head Contains general information about an HTML page that isn't displaed on the page itself.
target Target Specifies where a hyperlink should be opened
html HTML The root of an HTML document. Should be added after the !DOCTYPE declaration.
title Title Contains a text that defines the title of an HTML document.
li List Creates list items inside ordered lists and unordered lists.
video Video Embeds a media player for video playback.
em Emphasis Emphasizes text and browers will usually italicize the emphasized text by default.
ol Ordered List Creates a list of items in sequential order with numbers.
div Div Used as a container that divides an HTML document into sections and is short for "division"
br Line Break Creates a line break in text and is especially useful where a division of text is required.
img Image Embeds images in documents.
h1-h6 Heading Six different levels of heading elements.
p Paragraph Contains and displays a block of text.
ul Unordered List Used to create a list of items in no particular order.
alt Alternate Attribute Provides alternative text thats displayed if an image fails to render.
body Body Represents the content of an HTML document.
span Span An inline container for text and can be used to group text for styling purposes.
strong Strong Highlights important, serious, or urgent text and browers will normally render this highlighted text in bold by default.
